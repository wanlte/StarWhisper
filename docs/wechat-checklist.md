# 微信小程序审核 Checklist

## 内容合规
- [ ] 所有页面底部有「仅供娱乐 · 不涉及迷信内容」免责声明
- [ ] 首次进入弹出免责弹窗，用户需点击确认
- [ ] 无「算命」「吉凶」「祸福」「改运」「转运」等违规词汇
- [ ] 角色匹配结果标注为「虚构角色」「纯娱乐」
- [ ] 星座解读使用「性格趣味解读」「日常心情参考」等表述

## 功能合规
- [ ] 不涉及用户隐私数据收集（不索要手机号、身份证等）
- [ ] 无用户生成内容（UGC）功能，无需内容审核机制
- [ ] 无社交分享违规（仅使用系统剪贴板复制）
- [ ] 不包含诱导分享、诱导关注逻辑

## 技术合规
- [ ] 小程序包体积 < 2MB（不含本地图片则远小于2MB）
- [ ] 图片资源经压缩优化
- [ ] 无第三方不合规插件
- [ ] HTTPS 接口调用（生产环境）
- [ ] 小程序 AppID 已配置

## 视觉与体验
- [ ] 所有交互元素有触控反馈
- [ ] 触摸目标 >= 44px (移动端)
- [ ] 导航栏颜色/字体符合设计规范
- [ ] 不包含自动播放音频/视频

## 发布前检查
- [ ] 微信开发者工具「代码质量」检查通过
- [ ] 体验版在真机测试通过
- [ ] 类目选择：生活服务 > 休闲娱乐
